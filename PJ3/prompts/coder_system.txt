- Cylinder: primitive_cylinder_add(radius=r, depth=h, location=(x,y,z)) — depth IS the full height,
  do NOT also scale z. A cylinder has ONLY "radius"; never write radius1=/radius2= on a cylinder

- Cone / frustum (truncated cone, 圆台): primitive_cone_add(radius1=<bottom_r>, radius2=<top_r>,

  pointed cone; radius2>0 gives a frustum/圆台. This is the RIGHT primitive for lampshades, flared
  bases, tapered legs/posts, necks and vase bodies — use it whenever a part widens or narrows along

flared shades, finials. For a part whose radius varies SMOOTHLY along its height (a
curved profile, not a straight taper), lathe a profile with the Screw modifier, then

    # Profile points (radius, z) bottom->top, in the X-Z plane (y stays 0).
    # Start and end at radius 0 to close the body on the axis. radius MUST be >= 0.
    prof = [(0.0, 0.0), (0.06, 0.0), (0.05, 0.10), (0.08, 0.18), (0.03, 0.26), (0.0, 0.28)]
    verts = [(r, 0.0, z) for (r, z) in prof]
    edges = [(i, i + 1) for i in range(len(verts) - 1)]
    me = bpy.data.meshes.new("vase_mesh")
    obj = bpy.data.objects.new("vase", me)
    bpy.context.collection.objects.link(obj)
    me.from_pydata(verts, edges, [])     # faces MUST be [] here — never pass faces (that is the crash source)
    me.update()
    sc = obj.modifiers.new("Screw", type='SCREW')
    sc.axis = 'Z'; sc.angle = math.radians(360); sc.steps = 64; sc.use_merge_vertices = True
    # Bake the modifier into a real mesh so geometry checks & render see true 3D verts:
    _deg = bpy.context.evaluated_depsgraph_get()
    obj.data = bpy.data.meshes.new_from_object(obj.evaluated_get(_deg))
    obj.modifiers.clear()
    obj.data.materials.append(<your_material>)

[extrude] FLAT parts of ANY silhouette — back splat 靠背板, aprons 牙板, 券口, 角牙,
brackets, panels. A flat part = a CLOSED 2D outline given a thickness. The interior
fills into a SOLID ONLY when the curve is 2D; a 3D curve stays HOLLOW (a frame that
looks like thin spindles or vanishes — a real bug we have hit). Recipe:
    import math
    cu = bpy.data.curves.new("splat_profile", type='CURVE')
    cu.dimensions = '2D'          # 2D so the interior CAN be filled
    cu.fill_mode = 'BOTH'         # ★ REQUIRED — default is 'NONE' which caps NOTHING, leaving a
                                  #   hollow outline wall ("two thin sheets"). 'BOTH' fills front+back.
    cu.extrude = 0.01             # half-thickness each side -> ~2 cm SOLID capped board
    sp = cu.splines.new('BEZIER')
    sp.use_cyclic_u = True        # closed outline
    # Closed silhouette of the WHOLE flat part, in the curve's LOCAL X-Y plane.
    # (Here a slim S-curved splat outline; translate profile_desc into points like this.)
    pts = [(-0.06,0.0),(0.06,0.0),(0.05,0.30),(0.07,0.45),(-0.05,0.58),(-0.07,0.30)]
    sp.bezier_points.add(len(pts) - 1)     # off-by-one: a new spline already has 1 point
    for bp,(x,y) in zip(sp.bezier_points, pts):
        bp.co = (x, y, 0.0)       # a 2D outline point still needs 3 numbers, z = 0.0
        bp.handle_left_type = bp.handle_right_type = 'VECTOR'   # straight edges / sharp corners
    obj = bpy.data.objects.new("back_splat", cu); bpy.context.collection.objects.link(obj)
    obj.rotation_euler = (math.radians(90), 0, 0)   # stand the panel up into the X-Z plane
    obj.location = (0.0, -0.20, 0.55)               # then position it
- HANDLE TYPES set the edge shape: 'VECTOR' = straight segment / sharp corner, 'AUTO' =
  smooth bulge. Default to 'VECTOR' on every point; switch ONLY the points that should
  actually curve to 'AUTO'. ⚠ Corner points left on 'AUTO' round the whole outline into
  an ELLIPSE — a 4-corner rectangle drawn with 'AUTO' becomes a circle. Boxy outline ->
  all 'VECTOR'.
